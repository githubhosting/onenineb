Installation

Follow the steps below to setup your Vite project:

  1. Unzip the downloaded package and open the biteezy-cryptocurrency-landing-page-template-source-code folder to find all the template files..
  2. Open your terminal
  3. Go inside biteezy-cryptocurrency-landing-page-template-source-code
  4. Now install yarn in this folder Run Command yarn to install all dependencies yarn or npm install
  5. after installing all the dependencies a node modules folder is generated automatically. which contains all the necessary packages which we need to run this app.
  6. Firstly you need to add images as we required in this app
    Go inside biteezy-cryptocurrency-landing-page-template-source-code/src/assets/img Now Add Images in this folder with the same name
    • bitcoin.png - for hero section image
    • c1.jpg - for testimonials section client image
    • c2.png - for testimonials section client image
    • c3.jpg - for testimonials section client image
    • explore.png - for explore section image
    • logo.png - for main logo
  7. Now you can run your App open terminal and run the command yarn dev or npm run dev
  8. Below is the folder structure:
    • public - Contains all the public files
      • favicons - Favicons
    • src - Contains all the jsx files and source code for our project. Main folder
      • assets - contains all assets and images
      • components - contains all components
      • constants - contains all data in array format used in our app
      • server - Firebase configuration
      • styles - contains all stylesheet
      • App.jsx - Starting point
      • main.jsx - Our first main component which rendered
    • index.html - Main root (html) file
    • package.json - Contains details of all package and dependencies
    • package-lock.json - Contains all package and dependencies version details etc.
    • postcss.config.cjs - It Comes with Tailwind CSS
    • tailwind.config.cjs - Tailwind configuration file
    • vite.config.js - Vite configuration file
    • yarn.lock - automatic generated file for information about the current versions of dependencies
    • yarn-error.log - an error log
  9. You Only need to make necessary changes inside src folder as per your need.
  10. Build command! yarn build or npm run build

Firebase

Setup your firebase follow the below steps

Open Chrome browser And Go to this link to https://console.firebase.google.com/

  1. Now Login with your Gmail
  2. Click on create a project
    • Enter your project name
    • check all checkboxes
    • Click on continue
    • on next page unchecked the checkbox : Enable Google Analytics for this project
    • Click on Create project
    • After taking few seconds 20 to 30 seconds depends on your internet speed your project will be ready then click on contine

  3. Now on left sidebar you need to click on Build in build you need to click on Firestore Database Build/Firestore Database

    • Now click on create database
    • Start in production mode click on next
    • click on enable
    • if you choose start in production mode then you need to change some basic things In tabs click on rules and
      change allow read, write: if false; To allow read, write; click on publish

  4. Now click on sidebar on project overview You will see this type of interface
    • Now you need to click on this web icon </ >
    • Next you need to Add your app name
    • Click on Register app
    • Use npm Copy Firebase configuration code
    • note: you will required this firebase configuration code to run your app
    • Copy then click on button continue console
    • Now paste this firebase configuration code in biteezy-cryptocurrency-landing-page-template-source-code/src/Server/Firebase/config.jsx 
                const firebaseConfig = {
            apiKey: "YOUR_FIREBASE_API_KEY"
            authDomain: "YOUR_AUTH_DOMAIN",
            projectId: "YOUR_PROJECT_ID",
            storageBucket: "YOUR_FIREBASE_STORAGE_BUCKET",
            messagingSenderId: "YOUR_MESSAGE_SENDER_ID",
            appId: "YOUR_APP_ID"
          };
      In place of this paste your firebase configuration
    • Congrats.. Now you are ready to go.. your working contact form is Ready!!

What is Tailwind CSS?
Tailwind CSS makes it quicker to write and maintain the code of your application. By using this utility-first framework, you don't have to write custom CSS to style your application. Instead, you can use utility classes to control the padding, margin, color, font, shadow, and more of your application.
